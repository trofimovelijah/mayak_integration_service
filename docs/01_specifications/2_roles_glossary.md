# 2 Входная аналитика
В данном разделе представлены описание ролей и сервисов, чьё участие необходимо в достижении поставленной цели.

## 2.1 Роли

Ниже представлены роли, реализованные в рамках Платформы

1. Благотворитель (Донор):  
   1. отправка платежа на выбранное событие,  
   2. выбор периодичности отправки платежа (разово, ежемесячно),  
   3. ввод суммы платежа,  
   4. ввод имени и email для осуществления платежа.  
2. Пользователь Платформы:  
   1. регистрация на Платформе,  
   2. создание События,  
   3. отправка события на модерацию для размещения на Платформе “Давайте соберемся”,  
   4. отправка платежа на выбранное событие.  
3. Администратор:  
   1. принятие решения по модерации события, добавленным Пользователем,  
   2. просмотр статистики,   
   3. отключение (временное) сценариев,   
   4. изменение текста уведомлений,   
   5. ручная переотправка уведомлений (при необходимости).

## 2.3 Глоссарий

| Термин | Определение |
| ----- | ----- |
| Админ-панель | закрытый раздел Платформы для управления контентом, настройками и отображением статистики |
| Администратор | физическое лицо, имеющее доступ к специализированным возможностям Платформы через Админ-панель и являющееся должностным Фонда, либо выполняющее обязанности по поручению Фонда |
| Благотворитель | физическое лицо, который может отправлять платежи через Платформу на выбранные им События |
| Донор | см. Благотворитель |
| Заказчик | представители Фонда |
| Интеграционный сервис | см. Сервис интеграции |
| Модерация | процедура рассмотрения созданных Пользователями Событий для последующего размещения их на Платформе для проведения сборов |
| Подписка | см. Рекуррентный платёж |
| Пользователь | физическое лицо, достигшее 18 лет, имеющее доступ к Платформе посредством сети Интернет и размещающее предложения о Сборе через создаваемые События |
| Платформа | программно-аппаратный комплекс на базе 1С-Битрикс в сети Интернет по адресу [https://join.mayak.help/](https://join.mayak.help/)  |
| Пожертвование | дарение денежных средств Благотворителем через Платформу посредством выбранного События |
| Рассылка по электронной почте | отправка презентационных, маркетинговых и иных материалов о работе Фонда и Платформы Пользователям или Благотворителям, подтвердившим согласие на получение материалов |
| Работник по коммуникациям | специалист Фонда (либо специалист, которому Фонд делегирует подобные права), который занимается отправкой писем Благодарителям и модерирует входящие от Пользователей События |
| Рекуррентные благотворители | Благотворители, которые сделали выбор в пользу рекуррентных платежей. "статусы по рекуррентам" – к информации о состоянии их подписок. |
| Рекуррентный платеж | возможность выполнять регулярные списания денег с банковской карты покупателя без повторного ввода реквизитов карты и без участия благотворителя для инициации очередного платежа |
| Сбор (фандрейзинг) | сбор средств или ресурсов для организации или проекта в рамках благотворительных мероприятий на Платформе посредством использования Событий |
| Сервис интеграции | он же Интеграционный сервис – сервис в рамках Платформы, реализующий взаимодействие с внешними сервисами, находящимися вне юрисдикции Фонда, и внутренними составляющими, такими как 1С-Битрикс |
| Событие | размещенное на Платформе информационное объявление Пользователя, адресованное неограниченному кругу лиц \- посетителей Платформы и потенциальных Благотворителей |
| Статусы по рекуррентам | информация о состоянии Подписок Благотворителей |
| Триггер | автоматический запуск какого-либо действия при возникновении определенного события |
| Фонд | Благотворительный фонд «Дом с маяком», доступный по адресу [https://join.mayak.help/](https://join.mayak.help/)  |
