# 1 Описание проекта

## 1.1 Название проекта

Название проекта “_Интеграция платформы волонтерского фандрайзинга с внешними сервисами_”

## 1.2 Цель

Выстроить систему автоматической коммуникации с благотворителями и пользователями платформы через интеграцию в работу Фонда “**Давайте соберемся**” сервиса *Unisender Go* и внешних платёжных систем (CloudPayment, MixPlat) для постоянной и эффективной коммуникации с благотворителями.

## 1.3 Результат проекта

В результате реализации проекта должна быть реализована следующая функциональность:

* персонализированные письма по ключевым событиям,  
* прозрачная статистика коммуникаций,  
* автоматизация коммуникации с благотворителями и пользователями Платформы,  
* повышение вовлеченности и лояльности доноров,  
* исключение ручного участия работников в отправке писем.
