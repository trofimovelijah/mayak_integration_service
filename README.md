# mayak_integration_service
Работы по анализу нововведений для сервиса интеграций Платформы "Давайте соберёмся"

## 📚 Документация

Полная документация находится в папке [`docs/`](./docs/):

- **[Спецификация требований](./docs/specifications/SRS.md)**
- **[Диаграммы и артефакты](./docs/diagrams/)**
- **[Конфигурации](./docs/config/)**
- **[API](./docs/api/)**

## Структура

```bash
docs/
├── specifications/                    # SRS из Google Docs
│   └── SRS.md
├── diagrams/
│   ├── sequence/                      # PUML диаграммы последовательности
│   ├── bpmn/                          # BPMN диаграммы бизнес-процессов
│   ├── architecture/                  # Архитектурные диаграммы
│   ├── database/                      # ER диаграммы БД
│   └── api-flow/                      # API flow диаграммы
├── images/
│   ├── screenshots/                   # Скриншоты интерфейса
│   ├── ui/                            # UI/UX макеты
│   ├── diagrams/                      # Экспортированные PNG
│   └── icons/                         # Иконки
├── api/                               # API документация
│   ├── openapi.yaml
│   └── endpoints.md
└── README.md                          # Главный указатель
```
